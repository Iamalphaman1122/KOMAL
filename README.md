The Hotel Management System is a web-based application developed using HTML, CSS, and JavaScript. The main purpose of this project is to simplify hotel operations such as room booking, customer management, and billing.

This system allows users to log in securely, add new bookings, and manage guest details efficiently. Users can enter information such as guest name, room type, check-in and check-out dates, and the system automatically calculates the total cost based on the number of days stayed.

The application also includes features like searching bookings, editing and deleting records, and a dashboard that displays total bookings and revenue. All data is stored in the browser using Local Storage, so the information remains saved even after refreshing the page.

Additionally, the system provides a bill generation feature, allowing users to print invoices for customers. The design is responsive, making it usable on both desktop and mobile devices.

Overall, this project helps in reducing manual work, improving accuracy, and making hotel management more organized and efficient.
